TÀI LIỆU HỆ THỐNG API WMS - PHIÊN BẢN 2.0 (Toàn diện 123 APIs)
=============================================================

Hệ thống API đã được bao phủ hoàn toàn theo thiết kế chuẩn với 4 nhóm Prefix:
/erp/v1/*, /wms/v1/*, /integration/v1/*, /reports/v1/*
Ngoài ra có thêm /auth/v1/* và /mobile/v1/* để phục vụ chuyên biệt.
